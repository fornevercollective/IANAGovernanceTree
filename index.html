IANA Governance Tree

Mapping Internet Governance from IANA to Local Networks

About This Project

This project provides a structured model of internet governance from IANA down to local ISPs and servers, organized by Regional Internet Registries (RIRs): ARIN, LACNIC, RIPE NCC, AFRINIC, and APNIC. It helps visualize and interpret traceroute data across global jurisdictions.

Features

  • Hierarchical JSON database of IANA & RIRs
  • Traceroute parser and governance resolver tools
  • GeoIP and WHOIS integration
  • Governance tree visualizations
